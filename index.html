Collision Game

The Collision Game came from the "6.4 Assignment - Explore Collisions" project. This was my second favorite assignment for this class. In this assignment, we build a small game to help us understand collisions. we had to make a collision with either the mouse or keys. Once again I have never worked with gaming. This was very interesting to make and it was quite exciting to see how it came out and how it worked. Choosing the The number of objects moving was also something I was not expecting to be challenged with regarding what would look best. I was happy with how this assignment turned out.
